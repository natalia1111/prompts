Идеи для всех Естественно-научные предметы

Промпт 1. Сравнение объектов/систем

Сравни [укажите объекты или системы] по [укажите количество] параметрам, оформи в виде таблицы [укажите критерии сравнения].

Промпт 2. Инфографика “Как работает...”

Составь текст для инфографики “Как работает [укажите тему]” простым, понятным языком и с чёткой структурой.

Промпт 3. Эволюционный комикс

Создай сюжет комикса на тему “Как появился [укажите орган/признак/объект/систему]” с сохранением эволюционной логики и научной точности.

Промпт 4. Естественно-научный квест

Создай квест из [укажите количество] станций по теме [укажите тему], где каждая станция – мини‑задание (загадка, схема, поиск ошибки).

Промпт 5. Стишки‑порошки

Придумай [укажите количество] коротких стишков‑порошков по теме [укажите тему] для использования в начале или конце урока.

Промпт 6. Естественно‑научный ребус

Создай [укажите количество] ребусов по теме [укажите тему], чтобы учащиеся отгадывали термины, явления или объекты.

Промпт 7. Мини‑исследование

Предложи школьное исследование по теме [укажите тему], которое можно выполнить за 7–10 дней с простыми измерениями.

Промпт 8. STEAM‑проект

Разработай STEAM‑проект по [укажите название учебного предмета] на тему [укажите тему] с этапами, продуктом, необходимым оборудованием и критериями оценивания.

Промпт 9. Объясни простыми словами

Объясни тему [укажите тему] так, чтобы её понял учащийся 5-го класса (простые слова, примеры, метафоры).

Промпт 10. Объясни научно

Объясни тему [укажите тему] языком старших классов (научно) – с терминами, схемами, примерами экспериментов.

Промпт 11. Две версии объяснения

Сделай два объяснения темы [укажите тему]: 1) простое, 2) углублённое, и оформи их так, чтобы их можно было использовать на разных параллелях.

Промпт 12. Биология + география

Создай задание, где нужно объяснить, как климат влияет на [выберите нужное: растения/животных/почвы] в разных регионах.

Промпт 13. Биология + химия

Составь задание, где нужно объяснить химическую природу [выберите нужное: ферментов/витаминов/дыхания] и их роль в живом организме.

Промпт 14. Биология + математика

Создай задачу, где нужно рассчитать [выберите нужное: скорость роста растения, изменение численности популяции или массу биомассы] за определённый период.

Промпт 15. Межпредметный эксперимент

Предложи эксперимент, объединяющий биологию, географию и химию, который можно выполнить в школе за 1–2 недели, с измерениями и отчётом.

Промпт 16. Эксперимент “Климат в банке”

Предложи эксперимент, моделирующий климатическую систему в банке: испарение, конденсация, осадки, круговорот воды, изменение температуры.

Промпт 17. Научный арт‑объект

Придумай арт‑объект, который объясняет научное явление [фотосинтез, круговорот воды, образование торфа] с поясняющими подписями.

Промпт 18. Графики и диаграммы

Предложи [укажите количество] идей графиков по теме [климат/рост растений/растворы/популяции] и опиши, какие данные на них можно отразить.

Промпт 19. Объясни тремя языками

Объясни тему [укажите название темы] тремя способами: 1) как биолог, 2) как географ, 3) как химик, показывая разные грани одного явления.

Для учителя химии Лаборатории, кейсы, задачи

Промпт 1. Лабораторная работа по теме

Составь подробную лабораторную работу для [напишите класс] класса по теме [вставьте название темы]. Включи: цель, задачи (образовательные, воспитательные, развивающие), оборудование, реактивы, технику безопасности, пошаговый ход работы, таблицу наблюдений, контрольные вопросы и задания, критерии оценивания лабораторной работы.

Промпт 2. Мини‑лаборатория без реактивов

Придумай безопасную лабораторную работу без химических реактивов по теме [вставьте название темы], используя только бытовые материалы. Включи: цель, задачи (образовательные, воспитательные, развивающие), оборудование, реактивы, пошаговый ход работы, таблицу наблюдений, контрольные вопросы и задания, критерии оценивания мини-лабораторной работы.

Промпт 3. Лабораторная с ошибками

Создай лабораторную работу по теме [вставьте название темы], в которой будет 8–10 ошибок. Учащиеся должны найти и исправить их.

Промпт 4. Урок с ошибками

Разработай подробный урок для [напишите класс] класса по теме [вставьте название темы], в котором будет 8–10 ошибок. Учащиеся должны найти и исправить их.

Промпт 5. Практическая работа

Составь подробную практическую работу для [напишите класс] класса по теме [вставьте название темы]. Включи: цель, задачи (образовательные, воспитательные, развивающие), оборудование, реактивы, технику безопасности, пошаговый ход работы, таблицу наблюдений, контрольные вопросы и задания, критерии оценивания практической работы.

Промпт 6. Демонстрационный опыт

Составь алгоритм проведения демонстрационного опыта для [напишите класс] класса по теме [вставьте название темы]. Включи: цель, задачи (образовательные, воспитательные, развивающие), оборудование, реактивы, технику безопасности, пошаговый ход работы, таблицу наблюдений для учащихся.

Промпт 7. Химическая загадка

Придумай 5 химический загадок в стиле “чёрного ящика” по теме [вставьте название темы], чтобы их можно было использовать на этапе актуализации знаний и первичном закреплении.

Промпт 8. Химический кейс

Создай кейс для учащихся: «В лаборатории произошел несчастный случай [опишите ситуацию]. Предложи 3 научно-обоснованных решения кейса».

Промпт 9. Расчетные задачи

Составь расчетную задачу для [напишите класс] класса по теме [вставьте название темы]. Опиши алгоритм ее решения и укажи ответ.

Промпт 10. Цепочки превращения

Составь цепочку превращения из [укажите количество] звеньев для [напишите класс] класса по теме [вставьте название темы]. Ответ дай в виде стрелочной записи. Приведи 3 разных варианта.

Промпт 11. Контрольные вопросы

Придумай [укажите количество] контрольных вопросов для закрепления темы [вставьте название темы] в [напишите класс] классе.

Промпт 12. Химический диктант

Придумай [укажите количество] заданий в формате «химический диктант» по теме [вставьте название темы] в [напишите класс] классе.

Промпт 13. Химический стенд‑ап

Придумай [укажите количество] химических шуток или каламбуров для урока по теме [вставьте название темы] в [напишите класс] классе.

Промпт 14. Ошибки в уравнениях

Создай [укажите количество] уравнений реакций с ошибками (неверные коэффициенты, продукты, тип реакции), чтобы учащиеся их исправили.

Промпт 15. Электронные конфигурации

Составь [укажите количество] заданий на определение электронной конфигурации элементов, с ответами и краткими пояснениями.

Промпт 16. Химия вокруг нас

Создай [укажите количество] примеров химии в быту (еда, чистящие средства, косметика, вода) и объясни, какие вещества и реакции там задействованы.

Для учителя биологии Клетка, генетика, адаптации

Промпт 1. История клетки

Создай художественно‑научный рассказ от лица клетки [укажите тип клетки], объясняя её строение и функции через “жизненный день”.

Промпт 2. Генетическая задача

Создай [укажите количество] генетических задач разного уровня сложности по теме [вставьте название темы] с подробным решением.

Промпт 3. Ошибки в генетике

Составь текст с [укажите количество] ошибками по теме наследственности, чтобы ученики нашли и исправили их, объяснив свою правку.

Промпт 4. Модель ДНК

Предложи [укажите количество] варианта создания модели ДНК из подручных материалов (бумага, проволока, пластилин и т.п.) для школьного проекта.

Промпт 5. Адаптации вида

Опиши адаптации животного [укажите вид] к среде обитания в формате: [укажите количество] структурные, [укажите количество] поведенческие, [укажите количество] физиологические.

Промпт 6. Сравнение видов

Сравни два вида [укажите названия] по [укажите количество] параметрам: среда, питание, размножение, адаптации, роль в экосистеме, взаимодействие с человеком и др.

Промпт 7. Ошибки в тексте

Создай текст на 150–200 слов с [укажите количество] биологическими ошибками по теме [вставьте название темы], чтобы учащиеся нашли и исправили их.

Для учителя географии Климат, экономика, маршруты

Промпт 1. География одного предмета

Создай текст “География одного предмета” про [укажите название предмета]: происхождение, страны‑лидеры, пути распространения, современное использование, экологические последствия.

Промпт 2. География одного продукта

Опиши путь продукта [укажите продукт или выберите нужное: кофе/какао/рис/нефть/торф] от места добычи/выращивания до потребителя. Добавь описание регионов и карты маршрута.

Промпт 3. География одного природного явления

Создай текст “География одного явления” про [укажите явление или выберите нужное: тайфуны/муссоны/засухи/ледники]: где возникают, как распространяются, какие регионы затрагивают.

Промпт 4. Задание по карте

Составь задание по работе с физической картой для [напишите класс] класса по теме [вставьте название темы] (найти объекты, определить координаты, описать рельеф).

Промпт 5. Ошибки на карте

Создай карту‑описание с [укажите количество] географическими ошибками (неверные направления, объекты, названия). Учащиеся должны найти и исправить их.

Промпт 6. Климатический профиль

Создай климатический профиль страны [укажите название]: температура, осадки, ветры, климатические пояса, влияние на хозяйство и население.

Промпт 7. Инфографика “Как формируется климат”

Составь текст для инфографики “Как формируется климат региона [укажите регион]”: широта, рельеф, океаны, воздушные массы, течения, подстилающая поверхность.

Промпт 8. Сравнение климатов

Сравни климат двух стран [напишите названия стран] по [укажите количество] параметрам: средняя температура, амплитуда, осадки, сезонность, влияние океанов, ветры, рельеф, широта.

Промпт 9. Демографическая пирамида

Создай описание демографической пирамиды для страны [напишите название] и объясни, что она показывает (старение, рост, стабильность населения).

Промпт 10. Экономическая карта

Составь текст для карты “Хозяйство страны [напишите название]”: отрасли, ресурсы, транспорт, экспорт, ключевые центры.

Промпт 11. Географический кейс

Создай кейс: “Город [укажите название] столкнулся с проблемой [опишите проблему]. Предложи [укажите количество] решения с географическим обоснованием (рельеф, климат, ресурсы, транспорт).”

Промпт 12. География катастроф

Создай обзор природных катастроф региона [укажите название]: причины, последствия, меры защиты и адаптации населения.

Промпт 13. Маршрут путешествия

Создай маршрут путешествия по стране [укажите название]: 5–7 точек, природные объекты, культурные особенности, климат, логистика.

Промпт 14. Туристическая инфографика

Составь текст для инфографики “5 причин посетить [укажите регион]” (природа, культура, климат, кухня, особенные места).

Промпт 15. Географический путеводитель

Создай мини‑путеводитель по природным зонам [напишите название материка] с описанием климата, растительности, животных и людей.

Промпт 16. География + биология

Создай задание, где нужно объяснить, как климат и рельеф влияют на [выберите нужное: растения/животных/почвы] в разных природных зонах.

Промпт 17. География + химия

Составь задание, где нужно объяснить химическую природу [выберите нужное: кислотных дождей/загрязнения воздуха/качества воды] в разных регионах.

Промпт 18. География + математика

Создай задачу, где нужно рассчитать [выберите нужное: расстояние, масштаб, плотность населения или скорость течения реки] для [укажите регион].
